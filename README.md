This project is an interactive Power BI dashboard built to analyze Blinkit's grocery sales data. The dashboard provides insights into sales performance, customer ratings, item distribution, outlet performance, and other key business metrics.
I created this project by following a YouTube tutorial to improve my Power BI skills and understand how to build professional business dashboards. The dataset used in this project was downloaded from the YouTube video's description, while I recreated the dashboard myself for learning purposes.
🎯 Objective
The main objective of this project is to:
Analyze overall sales performance.
Identify the best-performing outlet types.
Compare sales across different outlet locations.
Analyze customer ratings.
Study product categories contributing to sales.
Build an interactive dashboard for business decision-making.

1. Total Sales
Value: $1.20M
Purpose
Shows the total revenue generated from all grocery sales.
Why it is important
Measures overall business performance.
Helps track revenue growth.
Used as the primary business KPI.
2. Average Sales
Value: $141
Purpose
Shows the average sales generated per item.
Formula
Average Sales = Total Sales ÷ Number of Transactions
Why it is important
Helps understand average customer spending.
Useful for pricing analysis.
3. Number of Items
Value: 8523
Purpose
Displays the total number of products available in the dataset.
Why it is important
Measures product diversity.
Helps inventory management.
4. Average Rating
Value: 3.9
Purpose
Displays the average customer rating for products.
Why it is important
Measures customer satisfaction.
Identifies product quality.

Located on the left side.

Outlet Size
Item Type

Allows users to interactively filter the dashboard.
How it works

Purpose

Why it is used

Identify high-performing establishment years.

X-Axis
Outlet Establishment Year
Y-Axis

The line connects yearly sales values, making trends easier to identify.
